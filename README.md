# AngularUIBootstrapBundle

Symfony 2 bundle which provides Angular UI Twitter Bootstrap.

![SensioLabs Insight][1]

## Installation

Install with composer :
	
	$ composer require "innova/angular-ui-bootstrap-bundle" "1.*"
